## How to Add Photos:

1. **Prepare your photos:**
   - Open them in an image editor (Photoshop, GIMP, Paint, etc.)
   - Resize to the recommended dimensions
   - Export/save in JPG or PNG format
   - Name them exactly as shown: img1.jpg, img2.jpg, etc.

2. **Add to this folder:**
   - Copy your prepared photos into this `photos` folder
   - Make sure filenames match exactly (case-sensitive on some systems)

3. **Test the website:**
   - Open `index.html` in your browser
   - Verify all photos display correctly
   - Test the audio by clicking the music button

## Tips:

- Use consistent aspect ratios for gallery images
- Optimize images to reduce file size (compress without losing quality)
- Test on mobile devices to ensure responsive display
- High-quality images make a big difference in the final result!
